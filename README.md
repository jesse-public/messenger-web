# messenger-web

Web client for E2E encrypted messaging

## Development

Recommended:
  - [VSCode](https://code.visualstudio.com/Download)

Required:
  - [Git](https://git-scm.com/downloads)
  - [NodeJS](https://nodejs.org/en/download/)

**To build and run:**

```
npm i
npm start
```
The app can then be accessed at http://localhost:8080
